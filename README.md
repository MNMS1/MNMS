# Welcome to MNMS 

## What is this ? 
an NMS solution for mikrotik devices that seeks to bypass the limitations that exist when managing mikrotik devices at scale.

## What is our objective? 
Avoid using VPN tunnels to manage devices through the API service as would normally be done, instead we decided to build a solution based 100% on the use of the fetch tool and the router scheduler.


## What functionalities will MNMS have?

- Collection of metrics of all interfaces and resources of the device according to the time defined by the user

- Alerts based on the metrics collected, for example alerts could be triggered when the rate of an interface changes from 100mb to 10mb or when x temperature is exceeded on a device

- Management tasks can be carried out centrally and massively, such as firmware updates, reboots, user creation and much more.
